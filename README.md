# zzp-starter-kit
# MHM ZZP Starter Kit

A practical, ready-to-use toolkit for ZZP’ers (freelancers) to run their business professionally:
- Invoices & quotations templates
- Expense & VAT (BTW) tracking sheets
- Client tracking
- AI prompt pack for client communication

## What’s inside
- Templates (Docs / Sheets / XLSX)
- A structured PDF guide
- Ready-to-sell ZIP export

## Folder structure
- `product/` — final deliverables (PDF, versioning)
- `templates/` — invoices, quotations, trackers
- `ai-prompts/` — curated prompts for daily ZZP operations
- `sales/` — landing page copy, Gumroad/Etsy listings
- `assets/` — cover + mockups
